# Aviral Writes

This project powers a terminal-style website that automatically pulls and displays posts from my Substack via RSS. It acts as a quick interface for browsing my writing outside of Substack’s native UI.

RSS source: **[aviralwrites.substack.com](https://aviralwrites.substack.com)** (my actual Substack publication)  
App deployment: **[aviralwrites.vercel.app](https://aviralwrites.vercel.app)**

## Tech

- [Next.js](https://nextjs.org) — React framework
- Deployed on [Vercel](https://vercel.com)
